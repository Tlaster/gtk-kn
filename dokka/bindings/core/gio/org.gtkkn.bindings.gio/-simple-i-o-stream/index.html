SimpleIOStream

open class SimpleIOStream(pointer: <Error class: unknown class><<Error class: unknown class>>) : IOStream

GSimpleIOStream creates a class@Gio.IOStream from an arbitrary class@Gio.InputStream and class@Gio.OutputStream. This allows any pair of input and output streams to be used with class@Gio.IOStream methods.

This is useful when you obtained a class@Gio.InputStream and a class@Gio.OutputStream by other means, for instance creating them with platform specific methods as ../gio-unix/ctor.UnixInputStream.new.html (from gio-unix-2.0.pc / GioUnix-2.0), and you want to take advantage of the methods provided by class@Gio.IOStream.
